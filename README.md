# Pilates Nathalie Cadotte — Formation Peak Pilates®

Landing page pour la Formation Peak Pilates® de Pilates Nathalie Cadotte.

## Pages

- `index.html` — Landing page principale (CTA → WellnessLiving)
- `formulaire.html` — Version avec formulaire de contact
- `appel.html` — Version avec réservation d'appel (Calendly mock)
- `mobile.html` — Aperçu mobile (iPhone)

## Déploiement

Site déployé via GitHub Pages depuis la branche `main`.

## Pour passer en prod

1. Brancher le `<form>` de `formulaire.html` (Formspree / CRM)
2. Remplacer le mock Calendly de `appel.html` par le vrai embed
3. Remplacer le placeholder VSL par l'iframe YouTube/Vimeo
4. Ajouter les vraies photos (Noémie, studio, équipement)
